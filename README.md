## Установка
1. Установить Python 3
1. В корневой папке выполнить:
   ```
   pip install -r requirements.txt
   ```


## Использование
1. Создать файл `repos.txt`
1. Заполнить файл `repos.txt` списком ссылок на репозитории. Ссылки вида `ssh://git@git.some.server:7999/ABC/some-project.git`. Каждая ссылка с новой строки.
1. Создать файл `tasks.txt`
1. Заполнить файл `tasks.txt` списком задач из релиза. Каждая задача с новой строки.
1. В корневой папке выполнить:
      ```
      python .\run.py
      ```
1. Сгенерированные файлы искать в папке `out`
